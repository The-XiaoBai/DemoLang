# DemoLang

## INTRODUCTION

DemoLang is a simple interpreted programming language that implements a complete lexical analysis, syntax analysis, and interpretation execution process.

This project is mainly used to demonstrate the basic principles and implementation methods of programming language interpreters.

## FEATURES

- REPL (Read-Eval-Print Loop)
- Lexer, Parser, and Interpreter implementation
- Basic data types

## BUILD

1. Clone or download DemoLang repository.
   ```bash
   git clone https://github.com/The-XiaoBai/DemoLang.git
   cd DemoLang
   ```

2. Use cmake to generate make files.
   ```bash
   mkdir build && cd build
   cmake ..
   cmake --build .
   ```
